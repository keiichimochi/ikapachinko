# 🦑 イカパチンコ改造仕様書 v2.0
## タスク概要
既存のイカパチンコ（HTML/Canvas/JS）に以下3つの新機能を追加実装せよ。

---

## 【機能1】触手チューリップ（間接付き・気持ち悪い動き）

### 仕様
- チューリップが開いた瞬間、内部から**イカの触手が3〜5本**ニョキッと出現する
- 触手は**3〜5個の間接（Joint）で構成**されたセグメント式
  - 各間接はサイン波＋ノイズで**独立してうねる**
  - 先端に向かうほど細くなる（根元 12px → 先端 3px）
  - 先端には**吸盤の丸**を描画（半径 4px の円、内側に小さい円）
- **動きの仕様**
  - 出現時：根元から順番に間接が伸びる（0.1秒ずつ遅延）
  - 通常時：ゆらゆらと不規則にうねる（気持ち悪さMAX）
  - 玉が半径80px以内に接近：触手が玉に向かって伸びる（追跡モード）
  - 玉に触れた瞬間：触手が巻きつくアニメ（0.3秒）→ 玉の軌道を30〜90度ランダムに曲げる
  - 収縮時：先端から順番に縮む
- **色** : グラデーション（根元 `#8B4513` → 先端 `#FF6B35`）、吸盤は `#FF4500`
- **効果音トリガー** : `playSound('tentacle_grab')` を玉接触時に呼ぶ

### コメント表示
触手出現時に以下をランダムで画面上部に表示（2秒間、フォントサイズ28px、黄色）:
```
"あーっ！チューリップから謎の触手が伸びてきた！"
"な、なんだこの触手はーー！！"
"触手に絡まれた！軌道が変わるぞ！"
"き、気持ち悪い…でも目が離せない！"
"Kiyoshi：「……フッ。これが俺の海だ。」"
```

## 【機能2】吸盤風車（玉を吸い込んで発射）

### 仕様
- 既存の**風車（windmill）を吸盤風車に改造**する
- **見た目**
  - 羽根の表面を吸盤テクスチャに変更
  - 各羽根に吸盤を等間隔で配置（羽根1枚につき3〜4個）
  - 吸盤：半径6pxの円、内側グラデーション（`#CC2200` → `#FF6666`）、縁取り `#880000`
  - 回転中は吸盤がぬるぬる動いて見えるようにする（回転アニメ）

- **吸い込みメカニズム**
  - 風車の中心から半径 **60px 以内**に玉が入ると吸引力が発生
  - 吸引力：玉を中心に向かって加速（`vx += (cx - ball.x) * 0.08, vy += (cy - ball.y) * 0.08`）
  - 中心半径 **15px 以内**に到達：玉を「吸収」状態にする（一時的に非表示）
  - 吸収後 **0.5〜1.0秒**（ランダム）で風車の回転方向に**射出**
  - 射出速度：吸収前の速度 × 1.5〜2.5倍（ランダム倍率）
  - 射出角度：その時の風車の回転角度に依存（予測不能にする）

- **演出**
  - 吸い込み中：風車の回転速度が一時的に上がる
  - 射出時：白いフラッシュエフェクト（0.1秒）
  - 最大吸収玉数：同時に3個まで（4個目は弾かれる）

### コメント表示
吸い込み発生時にランダム表示:
```
"吸盤風車が玉を吸い込んだ！どこに飛ぶ！？"
"ぐるぐる〜！吸盤パワー全開！"
"Kiyoshi：「テキーラと同じだ。吸って、吐く。」"
"風車が玉を飲み込んだ！！"
"これが気仙沼の渦潮だァ！！"
```

## 【機能3】モニタースロット（動画＆コメント表示）

### 仕様
- 既存のスロット枠（`UFO SLOT`エリア）を**モニター風UI**にリデザイン
- **モニター外枠**
  - 角丸矩形（`borderRadius: 8px`相当）
  - 枠色：`#00FF88`（ブラウン管グリーン風）
  - 内側：`#001100`（暗い黒緑）
  - スキャンライン効果：横線を半透明で重ねる（`rgba(0,255,0,0.03)`、4px間隔）
  - ちらつきエフェクト：0.5〜3秒間隔でランダムに輝度が微変動

- **動画エリア（上部60%）**
  - 実際の `<video>` タグは使わず、**Canvas上でアニメーション映像を生成**
  - 表示パターン（状況に応じて切り替え）:

  | 状況 | 映像内容 |
  |---|---|
  | 通常時 | イカが泳ぐピクセルアニメ（8フレームループ） |
  | リーチ | Kiyoshiの目がドアップ（ゆっくりズームイン） |
  | 大当たり | テキーラボトルが爆発するアニメ |
  | 触手発動中 | 触手がうねる映像（気持ち悪いやつ） |
  | ハズレ | 宮城の海の波（静かに揺れる） |

- **コメント欄（下部40%）**
  - ニコニコ動画風の**流れるコメント**を実装
  - コメントは右から左にスクロール
  - フォントサイズ：14px、色はランダム（明るい色のみ）
  - 常時5〜8件のコメントが流れる

  **通常時コメントプール:**

## 技術要件
- **言語**：既存コードに合わせる（HTML5 Canvas + Vanilla JS 推定）
- **物理**：Matter.js または独自の簡易物理で統一
- **フレームレート**：60fps 維持を優先
- **モジュール分割**：
  - `tentacle.js` : 触手クラス（`Tentacle`, `TentacleJoint`）
  - `suctionWindmill.js` : 吸盤風車クラス
  - `monitorSlot.js` : モニタースロット＋コメントシステム
  - `commentary.js` : 実況テキスト表示システム
- 既存の `START`, `ATTACK` ボタン、スロットロジックは**破壊しない**こと
- 新機能はすべて**既存クラス/関数を継承または拡張**する形で実装
