# Sound Effects

All files in this directory are short procedural WAV effects generated for the
Ikapachinko simulator. They are loaded through `AudioManager` in `index.html`.

## IDs And Files

| ID | File | Purpose |
| --- | --- | --- |
| `ui_click` | `ui_click.wav` | Button and control clicks |
| `ui_cancel` | `ui_cancel.wav` | Cancel or OFF toggle feedback |
| `ball_launch` | `ball_launch.wav` | Ball launch |
| `pin_hit_soft` | `pin_hit_soft.wav`, `pin_hit_soft_01.wav`, `pin_hit_soft_02.wav`, `pin_hit_soft_03.wav` | Light pin contact variants |
| `pin_hit_hard` | `pin_hit_hard.wav`, `pin_hit_hard_01.wav`, `pin_hit_hard_02.wav` | Hard pin contact variants |
| `wall_hit` | `wall_hit.wav` | Outer wall and bumper contact |
| `tulip_open` | `tulip_open.wav` | Tulip catch/open feel |
| `tulip_close` | `tulip_close.wav` | Tulip close feel |
| `tentacle_emerge` | `tentacle_emerge.wav` | Tentacle emergence |
| `tentacle_wiggle_loop` | `tentacle_wiggle_loop.wav` | Quiet tentacle loop |
| `tentacle_grab` | `tentacle_grab.wav` | Tentacle contact |
| `ball_deflect` | `ball_deflect.wav` | Tentacle deflection |
| `start_in` | `start_in.wav` | START catch |
| `slot_spin_start` | `slot_spin_start.wav` | Slot spin start |
| `slot_reel_stop` | `slot_reel_stop.wav`, `slot_reel_stop_01.wav`, `slot_reel_stop_02.wav`, `slot_reel_stop_03.wav` | Slot reel stop variants |
| `slot_near_miss` | `slot_near_miss.wav` | Near miss |
| `jackpot_small` | `jackpot_small.wav` | Small win |
| `jackpot_big` | `jackpot_big.wav` | Big jackpot / FEVER |
| `attack_ready` | `attack_ready.wav` | ATTACK ready |
| `attack_hit` | `attack_hit.wav` | ATTACK catch |
| `miss_drop` | `miss_drop.wav` | Ball drain |
