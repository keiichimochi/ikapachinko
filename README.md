# ikapachinko

UFOモチーフのブラウザ向けパチンコシミュレーターです。単体の `index.html` とローカルアセットで動作します。

## 起動

```bash
python3 -m http.server 8000 --bind 127.0.0.1
```

ブラウザで `http://127.0.0.1:8000/` を開きます。

## 主な機能

- 物理ベースのパチンコ玉挙動
- スロット大当り演出
- ガチャ動画演出
- `assets/bgm` のMP3によるBGMループ再生
- `assets/keihin` の3D景品を使ったバッジ/カード収集
- ハイスコアと収集状況のローカル保存
- パスワードなしの端末内保存
