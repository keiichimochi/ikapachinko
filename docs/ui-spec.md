# UIデザイン仕様

## 収集ラック

- 左側: バッジ10枠。
- 右側: カード10枠。
- 未獲得枠は暗いスロットと `?` 表示。
- 獲得済み枠は黄色の発光枠と景品名短縮表示。
- 新規獲得枠は短いポップアニメーションを再生する。

## 景品プレビュー

- 画面中央にモーダルとして表示する。
- `<model-viewer>` でGLBを表示し、回転とカメラ操作を許可する。
- 景品種別、枠番号、景品名、新規/重複状態を表示する。
- 閉じるボタン、背景クリック、または5秒経過で閉じる。

## レスポンシブ

- デスクトップではパチンコ台の左右にラックを配置する。
- モバイルではラックを横並びの折り返し表示に切り替える。
